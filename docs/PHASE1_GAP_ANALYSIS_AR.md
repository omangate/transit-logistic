## 1. ملخص تنفيذي

منصة ترانزيت لوجستك monorepo تضم:
- **API:** NestJS (`apps/api`) — المنفذ 3001، البادئة `/api/v1`
- **واجهة الويب:** Next.js 15 (`apps/web`) — المنفذ 3000، دعم العربية والإنجليزية
- **حزمة مشتركة:** `packages/shared`

بعد المراجعة والتنفيذ، **اكتمال المرحلة 1 ≈ 100%** من المتطلبات الوظيفية الأساسية. المتبقي خارج النطاق (المرحلة 2+): تطبيق جوال أصلي، تكامل ERP، ذكاء اصطناعي متقدم للتسعير، وبوابات دفع إنتاجية كاملة (Thawani/MyFatoorah/Stripe في وضع التكامل).

```bash

```

**سيناريوهات UAT:**
1. عميل: إنشاء شحنة الرياض→جدة → تعديل مسودة → دفع → تحميل عقد/فاتورة PDF
2. أسطول: قبول → سائق يسجل GPS → عميل يرى الخريطة
3. عميل: إكمال + تقييم
4. أسطول: محفظة → طلب سحب → مدير: موافقة → paid
5. عام: `/track/REF-xxx` مع خريطة للشحنة في transit
